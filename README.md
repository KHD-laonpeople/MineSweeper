# MineSweeper

### 1. 기능
  - 게임 보드 생성
    - Game Start 버튼 클릭
    - 입력한 보드 크기, 지뢰 개수에 맞는 게임 보드 생성
  - 좌클릭
    - 생성된 버튼 위에서 마우스 좌클릭
    - 버튼 오픈
    - 지뢰일 경우 게임 종료
    - 지뢰가 아닐 경우 주변 지뢰 개수 표시, 주변 지뢰가 없을 경우 주변 8칸 버튼 같이 오픈
  - 우클릭
    - 생성된 버튼 위에서 마우스 우클릭
    - 버튼 잠금, 깃발 표시
  - 랭킹
    - Game Start 버튼 클릭 시 시간 측정 시작
    - 게임 보드, 클리어 시간 측정을 통한 랭킹 5위까지의 기록 저장 및 출력
